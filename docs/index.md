# Thunder
The Deep Learning framework based on Lighnting

## Installation
You can install from pypi  
```bash
pip install thunder
```
or directly from GitHub  
```bash
git clone https://github.com/neuro-ml/thunder.git
cd thunder && pip install -e .
```
